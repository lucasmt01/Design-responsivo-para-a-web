Introduction

Python is a versatile, high-level programming language known for its simplicity and readability. It is a dynamically typed language and excels in various domains, from web development to data analysis and artificial intelligence.
Python comes with an extensive standard library that includes modules for data structures, mathematics, and system interaction. It offers core language elements such as operators, control structures, and statements.
Core Python can be extended using additional modules and libraries to cater to various purposes, for instance:

  • Client-side Python extends the core language by providing modules to interact with web browsers, manage web content, and handle user events.
  • Server-side Python extends the language by supplying modules relevant to server-side operations, facilitating database interactions, managing server resources, and handling file operations
Prerequisites

Before diving into Python, it's beneficial to have the following foundational knowledge:

  • A general understanding of programming concepts.
  • Familiarity with basic computer science principles.
  • Some experience with other programming languages will be helpful, but not mandatory.
Python vs. Other Languages

Python differs significantly from statically typed languages like C++ or Java. While Python shares some syntax similarities with these languages, it adopts a more dynamic approach.
In contrast to languages like Java that rely on class-based inheritance, Python uses a prototype-based object model. This model allows for dynamic inheritance, enabling objects to vary their inherited properties.
Python is known for its simplicity and readability. Unlike Java, it doesn't mandate declaring all variables or specifying variable types explicitly. It offers a more flexible and free-form syntax, allowing developers to focus on code logic rather than complex syntax.

Hello World

To begin with Python, let's write a simple "Hello World" program:

def greet_me(your_name): print("Hello, " + your_name) greet_me("World")

Run this code to see the output!

Variables

Variables in Python have scope defined by their declaration:

  • Variables declared outside any function are global and accessible throughout the entire script.
  • Variables declared within a function are local and confined to that function's scope.

Here is an example:

x = 10 # This is a global variable def my_function(): y = 20 # This is a local variable print(x) # Accessing global variable inside the function print(y) # Accessing local variable inside the function my_function()
Data Types

Python supports various data types:

  • Numbers: Integers, floats, and complex numbers
  • Strings: Sequences of characters
  • Booleans: True or False values
  • Lists: Ordered collections
  • Tuples: Immutable ordered collections
  • Dictionaries: Key-value pairs
  • Sets: Unordered collections of unique elements

Example:

# Different data types in Python my_integer = 42 my_float = 3.14 my_string = "Hello, Python!" my_boolean = True my_list = [1, 2, 3] my_tuple = (4, 5, 6) my_dictionary = {"key": "value"} my_set = {7, 8, 9}
Control Structures

Python provides control structures like:

  • If...else: Conditional execution
  • For loop: Iterating over sequences
  • While loop: Executing code while a condition is true

Example:

# If...else statement x = 10 if x > 5: print("x is greater than 5") else: print("x is less than or equal to 5") # For loop for i in range(5): print(i) # While loop y = 0 while y < 5: print(y) y += 1
Function Declarations

Functions in Python are defined using the def keyword followed by the function's name, arguments, and code enclosed within curly braces.

def my_function(parameter): print("This is a function with a parameter:", parameter) my_function("example")
